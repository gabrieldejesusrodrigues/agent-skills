# Agent Skills

Reusable AI agent skills for Claude Code and other AI agents.

## Installation

Install any skill using [skills.sh](https://skills.sh):

```bash
npx skills add gabrieldejesusrodrigues/agent-skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [startup-idea-validator](skills/startup-idea-validator/) | Multi-agent startup idea validation system that scrapes r/Startup_Ideas and evaluates ideas through Product, Business, and Devil's Advocate perspectives |
